# 限价单成交逻辑说明

## 功能概述

MockExchange的限价单成交逻辑已更新为基于**后续K线**判断是否成交，而不是基于历史K线。

## 核心逻辑

### 1. 成交判断规则

**买单（BUY）**:
- 如果订单价格 >= 后续某条K线的**最低价**，则成交
- 成交价格 = min(订单价格, 该K线收盘价)

**卖单（SELL）**:
- 如果订单价格 <= 后续某条K线的**最高价**，则成交
- 成交价格 = max(订单价格, 该K线收盘价)

### 2. 超时机制

- 检查后续**10条**1分钟K线
- 如果10条K线内都不满足成交条件，订单状态设置为 **EXPIRED**（失败）
- 一旦成交，立即停止检查后续K线

### 3. 示例说明

```
当前时间: 2023-01-01 00:00:00
当前价格: 16531.10

后续10条K线:
  1. 00:01:00 | 高:16531.20 低:16531.10
  2. 00:02:00 | 高:16531.10 低:16528.00
  3. 00:03:00 | 高:16528.10 低:16524.40
  ...
  10. 00:10:00 | 高:16515.70 低:16514.30

价格范围: 16514.30 - 16531.20
```

**场景1: 限价买单 价格=16522.75**
- 检查第1条K线: 16531.10 >= 16531.10? 否
- 检查第2条K线: 16522.75 >= 16528.00? 否
- 检查第3条K线: 16522.75 >= 16524.40? 否
- 检查第4条K线: 16522.75 >= 16521.60? **是** → 成交！
- 成交价格: min(16522.75, 16521.70) = 16521.70

**场景2: 限价卖单 价格=16696.51**
- 检查10条K线，最高价只有16531.20
- 16696.51 <= 16531.20? 始终为否
- 10条检查完毕，订单状态 → EXPIRED

## 代码实现

### 关键方法: `_check_limit_order_fill`

位置: [exchange_layer/mock_exchange.py:272-383](exchange_layer/mock_exchange.py#L272-L383)

```python
def _check_limit_order_fill(self, order: Order, max_bars: int = 10):
    """
    检查限价单是否成交（基于后续K线）

    逻辑：
    1. 获取当前最新K线的时间戳
    2. 获取后续的10条1分钟K线
    3. 逐条判断价格是否在高低价范围内
    4. 如果满足条件，更新订单状态为成交
    5. 如果10条内都不成交，标记为失败（EXPIRED）
    """
```

### 参数说明

- `max_bars`: 最多检查的K线数量，默认10条
- 可以在调用时自定义：`self._check_limit_order_fill(order, max_bars=20)`

## 订单状态

| 状态 | 说明 |
|------|------|
| NEW | 订单已创建（初始状态） |
| FILLED | 订单已成交 |
| EXPIRED | 订单超时未成交（10条K线内） |
| REJECTED | 订单被拒绝（下单失败） |

## 与实盘的差异

### 实盘行为
- 订单进入订单簿
- 等待对手方成交
- 可能部分成交
- 可以随时撤销

### MockExchange模拟
- 立即检查后续10条K线
- 全额成交或完全不成交
- 不支持部分成交
- 10条K线后自动过期

## 使用建议

1. **回测时**: 这个逻辑足够模拟真实的限价单成交情况
2. **参数调整**: 如果需要更长/更短的等待时间，可以调整`max_bars`参数
3. **价格策略**: 下单时应该参考当前市场价格，设置合理的限价

## 示例代码

```python
from exchange_layer import create_exchange, ExchangeType

exchange = create_exchange(ExchangeType.MOCK_LOCAL)
exchange.connect()

# 限价买单
buy_order = exchange.place_order(
    symbol='BTCUSD_PERP',
    side='BUY',
    order_type='LIMIT',
    quantity=1.0,
    price=16500.00  # 限价
)

print(f"订单状态: {buy_order.status.value}")
# FILLED - 成交
# EXPIRED - 超时未成交

exchange.disconnect()
```
